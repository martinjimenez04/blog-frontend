# Blog Frontend

React frontend for personal blog.

## Stack

- React
- React Router
- CSS

## Features

- Post list with preview
- Individual post view
- Responsive design
- Connected to FastAPI backend

## Installation
```bash
npm install
```

Create `.env`:
```env
REACT_APP_API_URL=http://localhost:8000
```

Start dev server:
```bash
npm start
```

Build for production:
```bash
npm run build
```

## Backend

This frontend connects to: [blog-backend](https://github.com/martinjimenez04/blog-backend)
